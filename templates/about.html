About - SewKinect

Custom Sewing Patterns, Custom Measurements

SewKinect was inspired by an absense of basic pattern generation software for sewers. Pattern drafting is key to custom-fit garments and costumes and relies on a geometric understanding three-dimensional body curves. Traditionally, patterns are drafted for individuals on paper at full scale, a huge inconvenience of time and space. SewKinect aims to streamline bespoke pattern development by creating a digital pipeline from measurements to printable pdf slopers.

SewKinect uses OpenNI/NITE to interface with a Microsoft Kinect. The Kinect is an afordable depth camera originally packaged with the XBox 360 for use with gesture-based gaming. While Microsoft supports a SDK for Kinect developers, it is currently Windows only.

OpenSource projects such as OpenNI aimed to increase access to Kinect development, but have since been discontinued. PyOpenNI and PyGame are used to power SewKinect's Kinect app.

Body measurements from the Kinect are posted to the SewKinect Flask server and then displayed along with the other options for pattern drafting.

Body measurements are calculated using a point cloud gathered the Kinect depth camera. By fitting ellipses to the depth information at coordinated corresponding to the Kinect joint skeleton, we can approximate the user's circumference.
